# Todo 리스트 기록 웹 📝

매일 해야 할 일을 기록하고 관리하는 Todo 리스트입니다.

## 기능

- ✅ 할 일 추가 (입력 후 '추가' 버튼 클릭 또는 Enter 키)
- ✅ 체크박스로 완료 표시 (취소선 표시)
- ✅ 삭제 버튼으로 항목 삭제
- ✅ 로컬 스토리지에 자동 저장 (앱을 다시 시작해도 데이터 유지)
- ✅ 반응형 디자인

## 사용 방법

1. 할 일을 입력하고 "추가" 버튼을 클릭하거나 Enter 키를 누르세요.
2. 체크박스를 클릭하면 할 일에 취소선이 표시됩니다.
3. 삭제 버튼(🗑️)을 클릭하면 항목이 삭제됩니다.
4. 모든 데이터는 브라우저의 로컬 스토리지에 자동으로 저장됩니다.

## 실행 방법

`index.html` 파일을 브라우저에서 직접 열면 됩니다.
